# Посади квітку в AR 🌸

Шкільний екологічний проєкт: «Не рви рослини — посади віртуальну квітку і милуйся!»

### Як користуватися:
1. Відкрий посилання на телефоні.
2. Дозволь доступ до камери.
3. Наведи камеру на підлогу (скануй поверхню).
4. Тапни по землі — з’явиться квітка, яка «росте».

### Технології:
- 8th Wall XR Engine (self-hosted, open-source 2026)
- A-Frame
- GLTF модель квітки
