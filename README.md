> Дотик до реальної поверхні – і виростає квітка. Ніяких даних, ніяких трекерів, тільки гра.

## 🧒 Для кого
Для дітей від 3 років під наглядом батьків. Додаток **не збирає**, **не передає** та **не зберігає** жодних даних.

## 📱 Як грати
1. Відкрийте сторінку на телефоні з підтримкою AR (Android 8+, iOS 14+).
2. Дозвольте доступ до камери.
3. Наведіть на будь-яку плоску поверхню (підлога, стіл).
4. Торкніться поверхні – з'явиться квітка.

## 🔒 Безпека
- ✅ Камера працює **тільки локально**, відео не записується.
- ✅ Немає аналітики, реклами, соціальних кнопок.
- ✅ Немає збору IP, геолокації, ідентифікаторів.
- ✅ Усі запити тільки до файлів репозиторію.
- ✅ Політика безпеки (CSP) блокує сторонні скрипти.

## 🛠 Розробникам
1. Клонуйте репозиторій.
2. Завантажте 8th Wall бінарник з [8th.io/xrjs](https://8th.io/xrjs) та покладіть `xr.js` і `xr-slam.js` у корінь.
3. Запустіть локальний сервер: `python -m http.server` або `npx serve`.
4. Відкрийте `http://localhost:8000` на телефоні в одній мережі.
