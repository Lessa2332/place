# Посібник для батьків

## Що робить цей додаток?
Ваша дитина може "садити" віртуальні квіти на реальних поверхнях через камеру телефону. Це розвиває просторову уяву та дає екологічне повідомлення: "Не рви квіти, краще посади віртуальну".

## Чи безпечно?
**Так, дотримано всіх норм 2026 року:**
- Жодних даних не збирається взагалі. Навіть ми, автори, не знаємо, хто і скільки грав.
- Камера активна тільки під час гри, відео не записується і не передається.
- Немає реклами, внутрішніх покупок, посилань на сторонні сайти.
- Додаток не вимагає облікового запису.

## Як запустити без інтернету?
1. Скачайте ZIP архів репозиторію.
2. Розпакуйте на комп'ютері.
3. Перенесіть папку на телефон (через USB або AirDrop).
4. Відкрийте файл `index.html` у браузері (Chrome або Safari).
5. Дозвольте доступ до камери.

## Що робити, якщо квіти не садяться?
- Переконайтеся, що поверхня добре освітлена та має текстуру (не одноколірна стіна).
- Спробуйте іншу площину (підлога краще за стіл).
- Оновіть сторінку.
